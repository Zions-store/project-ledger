# Project Ledger

AI agent skills for project structure analysis, development logging, and engine reference. Compatible with opencode, Claude Code, Codex, Cursor, and other Copilot-compatible agents.

## Skills

### project-onboard (v1.1.0)

Auto-detect project type and generate AGENTS.md for AI agents. 11 rule packs: Unity, Unreal, Node.js, Python, Rust, Go, Java, C++, C#, Lua, General. Zero dependencies.

- **Trigger phrases**: "onboard", "analyze this project", "generate AGENTS.md"
- **Force type**: `--type unity`, `--type unreal`, etc.

### project-docs (v1.0.0)

Initialize and maintain a three-document project knowledge system (`AGENTS.md` + `PROJECT_STATE.md` + `DEVLOG.md`). Auto-discovers type-specific templates (like project-onboard discovers rule packs).

- **Trigger phrases**: "init docs", "project-docs", "update project status"
- **Requires**: AGENTS.md (generated by project-onboard or manually created)
- **Template discovery**: `templates/<type>/PROJECT_STATE.md.tmpl` — create one, it auto-selects

### unreal-manual (v2.2.1)

Comprehensive UE5 reference for AI agents. Covers core concepts, C++/Blueprint patterns, GAS, MCP integration, delegates, smart pointers, async loading, and advanced gameplay patterns (2300+ lines).

- **Trigger**: Any Unreal Engine development question
- **Compatibility**: UE 5.0–5.8

### unity-manual (v2.1.1)

Comprehensive Unity reference for AI agents. Covers C# scripting, URP/HDRP, Shader Graph, Input System, game architecture patterns, mobile optimization, and profiling (1800+ lines).

- **Trigger**: Any Unity development question
- **Compatibility**: Unity 2022.3 LTS – Unity 6

### Workflow

```
project-onboard    →  AGENTS.md         (understand the project)
     ↓
project-docs       →  PROJECT_STATE.md  (track status, type-specific template)
                   →  DEVLOG.md         (record history)
     ↑
unreal-manual      →  engine reference  (how to build it)
unity-manual       →  engine reference
```

...and any other language the LLM can produce.

## Installation

Copy individual skill directories to your agent's skills folder.

### opencode
```
~/.config/opencode/skills/project-onboard/
~/.config/opencode/skills/project-docs/
~/.config/opencode/skills/unreal-manual/
~/.config/opencode/skills/unity-manual/
```

### Claude Code
```
~/.claude/skills/project-onboard/
~/.claude/skills/project-docs/
~/.claude/skills/unreal-manual/
~/.claude/skills/unity-manual/
```

### Other agents
```
~/.agents/skills/project-onboard/
~/.agents/skills/project-docs/
~/.agents/skills/unreal-manual/
~/.agents/skills/unity-manual/
```
